## Overview
This project presents a Discounted Cash Flow (DCF) valuation of Netflix, Inc. The objective of the model is to estimate the intrinsic value of the company by forecasting future cash flows and discounting them to their present value.

## Objective
The purpose of this analysis is to evaluate Netflix's fundamental value using a structured DCF approach and to gain practical experience in financial modeling and valuation techniques.

## Methodology
The valuation model follows these steps:

- Analysis of historical financial statements.
- Projection of key financial metrics over a 4-year forecast period.
- Estimation of Free Cash Flows.
- Calculation of the Weighted Average Cost of Capital (WACC).
- Determination of Terminal Value using the Gordon Growth Model.
- Discounting projected cash flows and terminal value to present value.
- Estimation of Enterprise Value and Equity Value.

## Key Assumptions

| Assumption | Value |
|-------------|--------|
| Forecast Period | 4 Years |
| WACC | 6.4% |
| Terminal Growth Rate (g) | 2.0% |
| Valuation Method | Discounted Cash Flow (DCF) |
