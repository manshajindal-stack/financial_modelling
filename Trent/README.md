# Trent Ltd. Financial Model & Valuation

## Overview

This repository contains a comprehensive financial model of **Trent Ltd.**, developed in Microsoft Excel as part of my financial modeling portfolio. The model is built using industry best practices and focuses on forecasting the company's financial performance and estimating its intrinsic value.

## Objectives

* Analyze Trent Ltd.'s historical financial performance.
* Forecast future financial statements based on key business drivers.
* Build an integrated three-statement financial model.
* Perform Discounted Cash Flow (DCF) valuation.
* Cross-check valuation using Comparable Company Analysis.
* Conduct sensitivity analysis to evaluate valuation under different assumptions.

## Model Components

* Historical Financial Statement Analysis
* Revenue & Operating Forecasts
* Integrated Three-Statement Model

  * Profit & Loss Statement
  * Balance Sheet
  * Cash Flow Statement
* Working Capital Schedule
* Depreciation & Capital Expenditure Forecast
* Debt Schedule
* Free Cash Flow to Firm (FCFF)
* Weighted Average Cost of Capital (WACC)
* Discounted Cash Flow (DCF) Valuation
* Comparable Company Analysis
* Sensitivity Analysis
