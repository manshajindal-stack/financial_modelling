# 📈 Eternal (Zomato) Revenue Scenario Builder

## Overview

This project is a driver-based revenue forecasting model built in Microsoft Excel to analyze how changes in key operating assumptions impact Eternal's projected revenue.

The model enables users to compare Base, Bull, and Bear case scenarios by adjusting business drivers through a centralized assumption panel.

---

## Model Highlights

- Driver-Based Revenue Forecasting
- Base, Bull & Bear Scenario Analysis
- Dynamic Assumption Control Panel
- Revenue Sensitivity Analysis
- Interactive Dashboard
- KPI Monitoring
- Revenue Growth Visualization

---

## Key Revenue Drivers

- Monthly Transacting Users
- Orders per User
- Average Order Value (AOV)
- Platform Take Rate
- Blinkit Growth
- Overall Revenue Growth Rate
